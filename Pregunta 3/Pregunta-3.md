###FrontPage Web:
Se considera un framework de CSS Boostrap que permita un diseño responsive
para la adaptacion de moviles y tablets. Tambien se considera la libreria
JQuery para la comunicacion con el Servidor Web.

###Web Service:
Se utiliza el servidor web Django para albergar la aplicacion web y gestionar
las peticiones web.

###API Gateway:
Se hace el uso del API gateway para gestionar y direccionar las distintas
peticiones del servidor web a los API's. Sirve de estacion central.

###API Social:
Se utiliza para gestionar las conexiones a los API's de las distintas 
redes sociales que se deseen conectar.

###API Sales:
Se utiliza para gestionar el sistema de venta de entradas y esta conectado 
a su vez con el API de verified by visa para asegurar las transacciones seguras.

###API Analytics:
Se utiliza para manejar toda la data relacionada al negocio de Cine Manzana y poder
generar estrategias a traves de los KPI's mas relevantes.

###API Catalog:
Es el eje central de la aplicacion de Cine donde se gestionan las carteleras,
cadenas de cines, peliculas y funciones. 

###Database Server:
Se opto por una BD No-Sql que sea capaz de adaptarse y moldearse a los distintos
requerimientos de cine manzana, ya sea la informacion del catalogo como de las 
ventas.
